**Hanabi-LLM Multi-Agent:**

Built a council-based multi-agent LLM system for the cooperative, imperfect-information game Hanabi, drawing on the LLM-Hanabi: [Evaluating Multi-Agent Gameplays with Theory-of-Mind and Rationale Inference in Imperfect Information Collaboration Game](https://www.alphaxiv.org/abs/2510.04980) benchmark (Liang et al., 2025). The approach decomposes decision-making into specialized agents for playing, hinting, and discarding, each performing probabilistic belief updates over hidden card states using Bayesian reasoning from public observations, hints, and discard information. It also implements deterministic greedy strategies such as maximizing clue efficiency and protecting critical cards to support controlled comparisons and strategy evaluation. An orchestrator coordinates agent execution, while a final decision agent aggregates structured recommendations and confidence scores to choose actions under uncertainty. The system is implemented in Python on DeepMind‚Äôs Hanabi Learning Environment with configurable OpenAI-compatible LLM backends.
